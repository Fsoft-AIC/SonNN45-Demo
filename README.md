# SonNN45 - Project Demos

Demo site for projects under **SonNN45**, affiliated with FPT Software AI Center, Vietnam.

Each project demo is hosted at:

```
https://fsoft-aic.github.io/SonNN45-Demo/projects/[project-name]
```

The source for each project can be found in the `projects/[project-name]` folder of this repository.

**Contact:** SonNN45@fpt.com
